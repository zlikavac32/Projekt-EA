Pojašnjenje opcija

Gradovi
koordinate gradova u formatu . Za gradove unesene kao pocetne vrijednosti, optimalna putanja je oko 7542
Sjeme
Cijeli broj koji se koristi kao sjeme za generator nasumičnih brojeva
Broj mrava
Određuje koliko se mrava sece
Algoritam
Odreduje koji se algoritam koristi za simulaciju. Moguce je odabrati izmedu sljedecih algoritama
  • Simple ACA
  • Ant System
  • Max-Min Ant System
Konstanta isparavanja
konstanta isparavanja koja odreduje kojim intenzitetom se obavlja isparavanje (u formulama oznaka )
Alfa
konstanta alfa (u formulama oznaka )
Beta
konstanta beta (u formulama oznaka )
Broj mrava azurira
Određuje koliko se mrava sa najboljim rjesenjem korisit za azuriranje feromonskih tragova
Broj setnji resetiraj
Određuje koliko setnji svakog mrava najbolje rjesenje mora stagnirati prije nego sto dode do resetiranja feromona
Broj setnji
Određuje koliko setnji ce svaki mrav napraviti
